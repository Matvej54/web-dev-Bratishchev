What is a box model in CSS

The box model in CSS defines how elements are rendered on a web page, treating them as rectangular boxes with content, padding, border, and margin. The box-sizing property determines whether an element's width and height include padding and border (`border-box`) or not (`content-box`).

What is a “clearfix hack” and when does it need to be applied

The "clearfix hack" is a CSS technique used to clear floats within a container element. It's applied when the container doesn't have a specified height and contains floated child elements, preventing layout issues like collapsed containers or overlapping content.

What is a “margin collapse” and what are the use cases for margin: auto and negative margin values

Margin collapse occurs when vertical margins of adjacent elements overlap, with the larger margin value being applied. margin: auto; is used to horizontally center an element within its parent.Negative margin values adjust element positioning, allowing for overlapping effects or fine-tuning layout.

What was the problem with achieving Holy Grail layout

The main problem with achieving the Holy Grail layout in CSS was creating equal-height columns, flexible column ordering, and ensuring compatibility across different browsers without using hacks or workarounds.

What are the modern ways to achieve Holy Grail layout

Modern ways to achieve the Holy Grail layout in CSS involve using Flexbox or CSS Grid, which provide cleaner and more intuitive solutions compared to traditional methods like floats and positioning.
